# Character Markdown Examples

This directory contains real-world examples of ESO character markdown output generated by CharacterMarkdown in GitHub format.

## Layout

Character profiles use **(account, location, slug)** identity:

```
examples/{account}/{location}/{slug}.md          # character profile (required)
examples/{account}/{location}/{slug}_plan.md   # build plan (optional)
examples/fixtures/                             # non-character docs (presets, crafting specs, tests)
examples/templates/                            # templates and schemas
```

- **account** — lowercase handle without `@` (`solaegis` for `@SOLAEGIS`)
- **location** — `na` or `eu` (megaserver)
- **slug** — lowercase underscore filename from display name (`Silent-Snow-Falls` → `silent_snow_falls`)

The same slug may exist under different accounts or locations, but not twice for the same account and location.

## Validation

Sync examples with ESO SavedVariables:

```bash
task examples:check
```

Audit drift without failing on orphan files:

```bash
task examples:check:audit
```

Validate example section list against LAM preset expectations (no ESO client required):

```bash
python3 scripts/validate_example_preset.py examples/solaegis/na/silent_snow_falls.md
python3 scripts/validate_example_preset.py --profile factory-defaults examples/solaegis/na/silent_snow_falls.md  # expect FAIL (not factory defaults)
```

### Preset vs factory defaults

Most `examples/solaegis/na/*.md` files are **GitHub format** exports from in-game `/markdown github`. They are **not** necessarily generated with **Reset to Defaults** — many match a LAM preset (e.g. **Solo PvE**).

| Profile | `silent_snow_falls.md` |
| --- | --- |
| **Factory defaults** ([`Defaults.lua`](../src/settings/Defaults.lua)) | Includes Inventory, Crafting Knowledge, Outfit Styles; no PvP, Quests, Armory, or Mail |
| **Solo PvE preset** | Includes PvP, Quest Progress, Armory, Mail, CP diagram, skill morphs, detailed Collectibles; omits Inventory / Crafting / Styles |

`silent_snow_falls.md` is the **Solo PvE** reference (paired with [`silent_snow_falls_plan.md`](solaegis/na/silent_snow_falls_plan.md)). Full toggle matrix: [`docs/settings-presets-matrix.md`](../docs/settings-presets-matrix.md).

When saving multi-chunk output, copy each chunk once in order; overlapping paste can duplicate sections (e.g. Collectibles + Companions).

## Generating your own

1. In-game, run: `/markdown github`
2. Copy the output from the window
3. Save as `examples/{account}/{location}/{slug}.md`

## Format details

All examples use the **GitHub format**, which includes:

- Collapsible sections (`<details>` tags)
- Rich tables with formatting
- UESP links for abilities, sets, races, classes
- Progress bars and emojis
- Badge-style headers with shields.io

For other formats (Discord, VS Code, Quick), see the main README.
