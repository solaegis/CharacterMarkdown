# Character Markdown Examples

This directory contains real-world examples of ESO character markdown output generated by CharacterMarkdown in GitHub format.

## Purpose

These examples demonstrate:
- **Complete character profiles** with all sections enabled
- **GitHub format** markdown with tables, collapsible sections, and UESP links
- **Real character data** from actual ESO characters
- **Formatting examples** showing how the output appears when pasted into GitHub

## Files

- `kellen_dysart.md` - Breton Sorcerer (Grand Sorcerer title)
- `korianthas.md` - Breton Arcanist (Bane of the Gold Coast title)
- `masisi.md` - Character profile example
- `pelatiah.md` - Character profile example

## Usage

These examples can be used to:
1. **Preview output** - See what your character markdown will look like
2. **Format reference** - Understand the structure and formatting
3. **Testing** - Compare your output against these examples
4. **Documentation** - Reference for users learning the addon

## Generating Your Own

To generate your own character markdown:

1. In-game, run: `/markdown github`
2. Copy the output from the window
3. Save it as a `.md` file

## Format Details

All examples use the **GitHub format**, which includes:
- Collapsible sections (`<details>` tags)
- Rich tables with formatting
- UESP links for abilities, sets, races, classes
- Progress bars and emojis
- Badge-style headers with shields.io

For other formats (Discord, VS Code, Quick), see the main README.
